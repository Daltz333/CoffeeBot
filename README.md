# CoffeeBot
Example Discord Bot programmed using the Java JDA

### Changing Token
* Application files are located under src -> main -> java
* Application constants are held in configurations -> Constants.java


### Running CoffeeBot
`.\gradlew run`

### Building CoffeeBot
`.\gradlew shadowJar`
